Function returns the values of input array by checking its overflow conditions.

Parameters
varray object to examine.
arrlenNumber of elements in an array.
allow_emptyTell whether array can empty or not.

Functions Used

get_typlenbyvalalign() : Given the type of OID return the value of typlen, typbyval, typalign.

Description

Function will check following conditions:

  • Return NULL if array is empty.
    if (allow_empty && (ndim == 0 || nitems <= 0)) {
    PGR_DBG("ndim %i nitems % i", ndim, nitems);
    return (int64_t*) NULL;
  • Return NULL if array is multi-dimension or no dimension.
    if (ndim != 1) {
    elog(ERROR, "One dimension expected");
    return (int64_t*)NULL;
  • Return NULL if no element found in an array.
    if (nitems <= 0) {
    elog(ERROR, "No elements found");
    return (int64_t*)NULL;
  • Validate input data type is ANY-INTEGER or not .

Store the value of an array in c_array and return.

Returns
Return the pointer of an array.
